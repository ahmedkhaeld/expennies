# Expennies
### Features
- User Authentication
    - Login/Registration
    - Password Reset
    - CSRF Protection
- Rate Limiting
    - Prevents brute force attacks
- Transaction Management
    - Create, Read, Update, Delete

- Dashboard

    - Total Balance

- Symfony components
